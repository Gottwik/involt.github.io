Getting Started - Quick start

Involt works like bridge between hardware and software which allows to create interaction by using web technologies. From software it's HTML with CSS similary to popular responisve design frameworks. The hardware part is reduced to minimum with Arduino which is best hardware prototyping platform. For basic interactions it's one line of HTML per one line of Arduino code. Currently Involt supports communication over serial and classic Bluetooth.

Involt from technical point of view does two things. It transfers variables between Arduino and app so they are accessible from both sides. Secondly, it contains UI kit generated by CSS classes and manipulated by JQuery.

The quick start section will show how fast and easy is Involt. More details are described in next chapters. The default settings are for serial communication.

How to install

Involt is created on top of Chrome API as offline Chrome Packaged App. It's easy to setup and suitable for prototyping purposes where Chrome dependency is not that important at all.

  1. Install Google Chrome.
  2. Download and unpack Involt.
  3. In Chrome go to tools > extensions.
  4. Toggle Developer mode
  5. Click "Load unpacked extension..." and choose unpacked Involt folder.

Now open your project with launch button or Chrome App Launcher.

CSS syntax

To create Involt HTML element you need to describe type of element and parameters. This is more explained in HTML UI & CSS syntax. This is important to understand before doing more things. Below there are examples of sending and receiving data.

ard [element] [pin] [value] [parameters]

Read data from Arduino

Reading data from Arduino is easy and requires two lines of code. Everything you need is the involtSend(pinIndex, value) function inside included Arduino sketch and HTML element you wish to communicate. Value of pin is stored and updated automatically in Involt.

...

void loop(){

involtReceive();


involtSend(0, analogRead(A0));

}

...

Now add this element to index.html file.

<div class="ard show A0"></div>

Send data to Arduino

Sending data requires digitalWrite or analogWrite with involtDigital[pinIndex] array value. The values in this array are updated automatically. Sending and receiving can be related to any variable you want - not only pin states. You can use this array to store values for sketch functions.

...

void loop(){

involtReceive();


analogWrite(5, involtDigital[5]);

}

...

The button will send pwm value 123 to pin 5 and turn on connected LED.

<div class="ard button P5 123">Click Me</div>

Summary

As you can see using Involt is easy to understand. There is more you can do with Involt. In next chapters there is extended description of basics and advanced functions. Starting playing with examples is good idea to practice.

Involt is ready to use Chrome packaged app template. You can use it to quickly create interactive object with software to hardware communication. The goal is to help designers to overcome the obstacles at prototyping stage.
