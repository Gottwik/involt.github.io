How to start with involt?

1. Install

  1. Install Google Chrome.
  2. Download and unpack Involt.
  3. In Chrome go to tools > extensions.
  4. Toggle Developer mode
  5. Click “Load unpacked extension...” and choose unpacked folder.
Now you can open your project with launch button or Chrome App Launcher.

2. Syntax

Involt translate CSS classes into specified UI element and function.
ard [function] [pin] [value*] [yourclass]

  1. [function]

    Describe what kind of object you want to create... check reference

  2. [pin]

    Analog or digital pin number. Type of pin define what kind of data you want to send.

    P - PWM pin data - P5 , P6 ...

    D - digital pin data (0/1) - D5 , D13 ...

    A - analog pin (read) - A0 , A1 ...

  3. [value]

    Number you want to send or define basic parameters like maximum value.

    *Some elements require two or more values to work.

  4. [yourclass]

    You can add your own class to element after the Involt specified classes.

3. Read data from Arduino.

//inside Involt Arduino sketch

...

void loop() {

chromeReceive();


chromeAnalog[0]=analogRead(A0);

}

...

...

5. Distribute data inside Arduino using array.

//inside Involt Arduino sketch

boolean directMode = false;

...

void setup() {

Serial.begin(115200);

pinMode(5,OUTPUT);

pinMode(6,OUTPUT);

}
void loop() {

chromeReceive();


analogWrite(5, chromeDigital[5]);

analogWrite(6, chromeDigital[5]);


chromeSend();

}
...

//inside index.html

<body>

<div class="ard button P5 123">Click Me</div>

</body>

6. Involt setup

  1. Where to start?

    To add content to your project edit index.html. When you make changes you need to reopen your app.

  2. Loader

    Every time you open your project Involt ask you for your Arduino port. You can customize loader screen in core/framework.css

    How to remove loader

    1. Open core/involt.js.
    2. Change "loaderOnLaunch" variable to false.
    3. Set "defaultSerialPort" to your connected Arduino port.

  3. User interface

    If you want to change Involt styleguide you need to edit core/framework.css. You can also create your own file to add custom CSS.

  4. How to change window dimensions

    Open core/background.js and change width/height.

  5. Name and description of project

    Open manifest.json. Do not change other elements inside manifest because your project will disappear from launcher.

7. Arduino communcation

  1. Arduino sketch setup

    1. Direct mode

      Direct mode send data from app to digital pins without storing them in chromeDigital array inside Arduino sketch. For more complex project with specified libraries and multiplexers you should set directMode to false and use chromeDigital array to distribute data.

      2. //if directMode=false
      void setup() {

      Serial.begin(115200);

      pinMode(6,OUTPUT);

      }
      void loop() {

      ...

      analogWrite(6,chromeDigital[6]);

      ...

      }

    2. Sending analog data

      chromeAnalog array contains each analog pin value. You can change their total number but remember to edit the "analogNumber" value and Involt.js.

    3. Send more variables to Arduino

      Set directMode to false and extend your chromeDigital array to store more values. After that you can send them to Arduino as digital pin that do not exist and use it as variable inside sketch.

  2. How to change total analog pin numbers or receive additional variables from Arduino

    1. Open core/involt.js

    2. To edit amount of analog pins change "analogPinsNumber".

      Remember to include them in "chromeAnalog" array inside Arduino sketch.

    3. If your project need more data to send you can change "additionalVariables" number.

      Example: You have all of your Arduino UNO 6 analog pins used but you need to send two more variables.
      Change "additionalVariables" to 2. Now you can send from "chromeAnalog" array two more variables as pins A6,A7.

8. JQuery & Chrome support

  1. How does it work?

    CSS classes contain specified function that JQuery converts to raw serial data (pin number, value). Involt automatically exchange data between Arduino and App using Serial library and Chrome API. You can use these values as data array inside Arduino without writing additional lines of code.

  2. You can add custom JQuery file to create more advanced project. Here is a list of framework variables:

    1. analogPins[k]

      Array that store analog pins value where "k" is pin number (pin A0: k = 0).

      2. //button that change font size based on pin A0 value
      $(".yourbutton").click(function() {

      $("div").css('font-size', analogPins[0]);

      });
      void loop() {

      ...


      int val = analogRead(A0);

      chromeAnalog[0]=map(val, 0, 1023, 1, 150);

      ...


      }

    2. digitalPins[pinNumber]

      Array that store digital pins. pinNumber is number without the first letter (D5 = 5, P5 = 5).
      //Display value of pin 5 when clicked.
      $(".yourbutton").click(function() {

      $(this).html(digitalPins[5]);

      });

    3. arduinoSend(pin, value);

      Use this function for sending data to arduino.

      Example:

      //Button that send to PWM pin 5 value 123
      $(".yourbutton").click(function() {

      arduinoSend("P5", 123);

      });

  3. To check what you can do with Chrome API go to:

    https://developer.chrome.com/apps/api_index
