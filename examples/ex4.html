Circuit

  1. This example shows how to prepare your custom Jquery script to control element without use of framework UI elements.

    Connect potentiometers to pin A0 & A1


  2. Add analogRead to your sketch.
    void loop() {

    chromeReceive();


    chromeAnalog[0]=analogRead(A0);

    chromeAnalog[1]=analogRead(A1);


    chromeSend();

    }

HTML & CSS

  1. In this tutorial we need to create big square at the center of app screen. First add HTML code to create square.
    2. <body>

    <div class="square"></div>

    </body>
  2. Create your custom Jquery script file and CSS in Involt folder. Name the new files customjquery.js and customstyle.css. Link them in head section.

    Important: You are not allowed to use inline scripting in Chrome App.

    3. <script src="customjquery.js"></script>
    <link rel="stylesheet" type="text/css" href="customstyle.css">
  3. Edit CSS
    4. .square{

    margin: 200px auto;

    width: 200px;

    height: 200px;

    background: gray;

    }
  4. We want real-time manipulation of our square. Use set interval to automatically update data. Knob value is stored in "analogPins" array.
    5. setInterval(function(){

    $(".square").css({


    "width": analogPins[0],

    "height": analogPins[0],

    "transform": "rotate("+analogPins[1]+"deg)"


    });

    },50);
