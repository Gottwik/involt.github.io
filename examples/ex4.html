Involt as JQuery plugin.
  1. Introduction

    By using JavaScript it's possible to create your custom scripts and UI elements. JQuery can be used not only to manipulate current Involt objects - it's easy to create new UI elenents and their interaction. With some advanced coding skills you can do most of your project with JS. Below there are examples of Involt JQuery methods. The best way to include your custom script is adding it in head section of index.html.

  2. Defining pin

    Target pin is most important part of framework - it's not only the pin we want to communicate with. Pin is index of digitalPins[] and analogPins[] global array which can be used to transfer more variables between software and hardware.

    .pinDefine() is required to communicate with framework and it should be your starting point. .pinDefine() adds to object pin and pinNumber data.

    Here is example how to attach pin data to object:

    $(this).pinDefine("P5");
  3. Sending the value

    There are many ways to send the value. To completly integrate with framework use the .sendValue() method. Remember about defining the desired value in digitalPins[] array - without doing this you can't easily communicate with generated Involt elements (they are depended on this array).

    Here are simple JQuery button examples:


    //by defining the value in array this can be used for other Involt elements to operate.

    $(".test1").click(function() {

    digitalPins[10] = 100;

    $(this).pinDefine("P10").sendValue();

    });

    //From the hardware point of view this is the same but value hasn't been updated.

    $(".test2").click(function() {

    $(this).pinDefine("P10").sendValue(100);

    });

    //Other way to send a value, arduinoSend(pin, value) is not based on JQuery

    $(".test3").click(function() {

    arduinoSend("P10", 100);

    });

    //More complex solution of arduinoSend

    $(".test3").click(function() {

    $(this).pinDefine("P10").updateValue(100);

    arduinoSend($(this).data("pin"), digitalPins[$(this).data("pinNumber")]);

    });

    To update the value without sending it use .updateValue(). This method changes the digitalPins[pinNumber] value where pinNumber is the defined pin number (P3 = 3).


    //update the value refered to target pin and send it

    $(".test4").click(function() {

    $(this).pinDefine("P11").updateValue(100).sendValue();

    });

  4. Interaction between elements

    To interact with other elements start with basic Involt button ard button P6 200.

    Use .pinSwap() to change pin of target element.


    //now the target pin of button is changed from P5 to P9

    $(".test5").click(function() {

    $(".button").pinSwap("P9");

    });

    //change the pin to P11 when double-clicked

    $(".test5").dblclick(function() {

    $(".button").pinSwap("P11");

    });

    Because .pinSwap() transfers the previous pin variable which simplify the exchange sometimes it's better to use again .pinDefine() when the value is defined or use this method with.updateValue().


    //Now there is new value attached to this button

    $(".test6").click(function() {

    $(".button").pinSwap("P11").updateValue(30);

    });

    //Selecting pin class is also good idea to operate between framework elements

    $(".test6").click(function() {

    $(".P6").pinSwap("P11").updateValue(30);

    });

    To send value attached to other element just write this code:

    $(".test7").click(function() {

    $(".button").sendValue();

    });

    It's possible to manually define or update object data with .data(). The pinDefine is equal to:

    $(this).data("pin", "P6");
    $(this).data("pinNumber", 6);
  5. Send multiple values to multiple pins

    It's easy to send one value to multiple pins because this requires only using the same chromeDigital[] value for each pin in Arduino sketch. The problem starts when sending multiple values to multiple pins at once.

    To do this there are many possibilities:

    //This is the simplest example

    $(".test9").click(function() {

    arduinoSend("P9", 10);

    arduinoSend("P10", 50);

    arduinoSend("P11", 100);

    });

    //You can use defined value from digitalPins[]

    $(".test9").click(function() {

    digitalPins[9] = 10;

    digitalPins[10] = 50;

    digitalPins[11] = 100;

    arduinoSend("P9", digitalPins[9]);

    arduinoSend("P10", digitalPins[10]);

    arduinoSend("P11", digitalPins[11]);

    });

    You can also combine multiple methods like this:

    $(".test10").click(function() {

    $(this).pinDefine("P9").sendValue(50).pinSwap("P10").sendValue(10);

    });

    For many data sent at once arduinoSend(pin, value) is better solution because JQuery is slower. Sending too much data can cause buffer overload and errors in communication (remember about this when using loops).

  6. Read data from arduino

    The values in analogPins[] array are updated automatically. Currently there are no methods related to data receiving but keep in mind that the only thing you need to do is to use this array in your JS.

Involt is ready to use Chrome packaged app template. You can use it to quickly create interactive object with software to hardware communication. It is aimed for UX/product designers rather than professional coders.
