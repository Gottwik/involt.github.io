Trigger function in Arduino with Involt element.
  1. Introduction

    This tutorial is for sending information to Arduino and call function defined in sketch. Involt is not passing the whole function from JavaScript. With custom script it's only possible to pass the results to chromeDigital[] or defined string. There is special attribute that allows sending additional string that on send can trigger something inside sketch. With this function name you can add your own function inside sketch called when it's received from app.

  2. Define the function name

    The function name can be defined as fn html attribute or with .sendFn() method. Now when the button is pressed there is sent pin value and additional name of your function.

    3. HTML

    <div class="ard button P6 200" fn="involt">Click me</div>

  3. Arduino sketch

    Inside sketch there is string called fname. It contains the attribute received from button. To prevent function from repeating the fname is cleared at the end of loop. It's possible to add the function directly in chromeReceive() but this is not recommended for beginners.

    4. ARDUINO

    chromeReceive();

    if (fname == "involt"){

    void involt();

    }

    fname = "";

    This tutorial is similar to defining custom string but in this case button sends at the same time additional string (and value).

  4. JQuery

    The .sendValue() method automaticaly checks if there is fn in object data. To send only the function use .sendFn().

Involt is ready to use Chrome packaged app template. You can use it to quickly create interactive object with software to hardware communication. It is aimed for UX/product designers rather than professional coders.
