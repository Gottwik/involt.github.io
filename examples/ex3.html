RGB slider

Circuit

  1. This example shows how to control RGB LED using three sliders.

    Connect LED with 220 ohm resistor to pins 3, 5 & 6.


  2. Now add pinMode to your void setup inside Arduino sketch. To control the values we will use directMode.
    boolean directMode = true;

    void setup() {

    Serial.begin(115200);

    pinMode(3,OUTPUT);

    pinMode(5,OUTPUT);

    pinMode(6,OUTPUT);

    }

Three lines of code to control your RGB LED.
