Circuit

  1. This example shows how to display values from two potentiometers.

    Connect potentiometers to pin A0 & A1


Display knob value.

  1. Add analogRead to your sketch.
    2. void loop() {

    chromeReceive();


    chromeAnalog[0]=analogRead(A0);

    chromeAnalog[1]=analogRead(A1);


    chromeSend();

    }
  2. Create HTML code to display values.
    3. <body>

    <div class="ard show A0"></div>

    <div class="ard show A1"></div>

    </body>
