2 Knobs 1 App
  1. Circuit

    1. This example shows how to display values from two potentiometers.

    Connect potentiometers to pin A0 and A1

  2. Display knob value

    1. Add analogRead to your sketch inside chromeSend function. The delay prevents the sketch from serial port overload.

    ARDUINO

    void loop() {

    chromeReceive();


    chromeSend(0, analogRead(A0));

    delay(1);

    chromeSend(1, analogRead(A1));

    delay(1);

    }

    2. To display values create HTML code for two different UI elements.

    HTML

    <body>

    <div class="ard show A0"></div>

    <div class="ard bar A1 1024"></div>

    </body>
