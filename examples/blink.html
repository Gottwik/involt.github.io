Circuit

This example explains how to connect single LED and how control it with buttons inside app. First of all connect LED with 220 ohm resistor to pin 5.

Basic example

Make sure that Involt is installed. If you don't know how to do this go to getting started page.

Add code to body section to generate switch between HIGH and LOW states of pin (the letter D in pin is for digital pins and P for PWM values or any other number). The last class is 0 because we want to start with inactive LED.

<div class="ard toggle D5 0"></div>

Now add digitalWrite to void loop and pinMode to void setup inside Involt Arduino sketch (in Arduino folder). The value received from App is stored in chromeDigital[] array with index which represents the target pin.

...
void setup{

Serial.begin(57600);

pinMode(5,OUTPUT);

}

void loop(){

involtReceive();


digitalWrite(5, involtDigital[5]);

}

...

Example with direct mode

Another example of LED blinking is two separate buttons with Direct Mode. This time we want to set it to PWM value. Change the pin number from D5 to P5. To toggle between two PWM values you can also use button-toggle button. In arduino sketch remove previous code and only change the directMode and autoPinMode to true. Now you don't have to change anything inside sketch to operate with basic interactions between pins (like blinking led). This mode is good for learning but not best for your project.

<div class="ard button P5 40">Turn me On</div>
<div class="ard button P5 0">Turn me Off</div>
