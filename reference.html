ToggleReturn to top

Toggle digital pin ON and OFF. [startvalue] is pin state on Arduino startup (need to be declared in void setup( ) ).

ard toggle [pin] [startvalue]
HTML
<body>

<div class="ard toggle D5 1"></div>

</body>
Arduino
void setup() {

Serial.begin(115200);

pinMode(5,OUTPUT);

digitalWrite(5,1);

}

IncreaseReturn to top

Increase pin value by [increment]. [maxvalue] is maximum limit of value. Works with decrease but both need to have same [startvalue].

ard increase [pin] [startvalue] [maxvalue] [increment]
HTML
<body>

<div class="ard increase P5 100 255 10"></div>

</body>
Arduino
void setup() {

Serial.begin(115200);

pinMode(5,OUTPUT);

analogWrite(5,100);

}
