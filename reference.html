Show

Display the analog pin value as number.

SYNTAX
ard show [pin]
HTML

<div class="ard show A0"></div>
ARDUINO

chromeSend(0, analogRead(A0));

Toggle

Toggle digital pin ON and OFF. [startvalue] is pin state on Arduino startup (need to be declared in void setup( ) ). The pinMode is requied to correctly toggle between digital values. For switching between two PWM values use toggle-pwm.

SYNTAX
ard toggle [pin] [startvalue]
HTML

<div class="ard toggle D5 1"></div>
ARDUINO

//remember about chromeDigital[] array in non-direct mode
void setup() {

Serial.begin(115200);

pinMode(5,OUTPUT);

digitalWrite(5,1);

}

...

Toggle-pwm

Toggle PWM pin between [value1] and [value2]. [value1] is also pin state on Arduino startup (need to be declared in void setup( ) ).

SYNTAX
ard toggle-pwm [pin] [value1] [value2]
HTML

<div class="ard toggle-pwm P5 50 100"></div>
ARDUINO

//remember about chromeDigital[] array in non-direct mode

Serial.begin(115200);

pinMode(5,OUTPUT);

analogWrite(5,50);

Increase

Increase pin value by [step]. [maxvalue] is maximum limit of value. Works with decrease but both need to have same [startvalue]. You can add in void setup the [startvalue] (this is not required).

SYNTAX
ard increase [pin] [startvalue] [maxvalue] [step]
HTML

<div class="ard increase P5 100 255 10"></div>
ARDUINO

//remember about chromeDigital[] array in non-direct mode
void setup() {

Serial.begin(115200);

pinMode(5,OUTPUT);

analogWrite(5,100);

}

...
