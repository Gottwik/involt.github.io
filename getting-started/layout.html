Layout and Pagination

To save some time on setting up basic pagination system Involt comes with mobile-like navigation elements.

Single-page layout is required but using Involt pagination isn't. It's not a problem if you wish to use other plugins for pagination.

Main navigation tabs

The main navigation level consists of tabs and their pages. Links href parameter inside navigation (involt-header > involt-nav) coresponds to page attribute of your page (without # sign). The tabs can be arranged at the top or bottom of screen by adding bottom class to involt-nav.

<header class="involt-header">

<nav class="involt-nav">

<a href="#page-1" class="involt-pagelink">First</a>

<a href="#page-2" class="involt-pagelink">Second</a>

<a href="#page-3" class="involt-pagelink">Third</a>

</nav>

</header>

<div page="page-1" class="involt-page home">First page</div>
<div page="page-2" class="involt-page">Second page</div>
<div page="page-3" class="involt-page">Third page</div>

Page header

Add single header for all tabs:

The title can be title-center or title-left. For additional buttons with icons you should consider using icon fonts like font awesome.

<header class="involt-header">

<div class="involt-appbar"><div class="title-center">Involt</div></div>

<nav class="involt-nav">

links...

</nav>

</header>

Different header for each tab require multiple involt-appbar with defined page attribute inside involt-header.

<header class="involt-header">

<div page="page-1" class="involt-appbar"><div class="title-center">First</div></div>

<div page="page-2" class="involt-appbar"><div class="title-center">Second</div></div>

<div page="page-3" class="involt-appbar"><div class="title-center">Third</div></div>


<nav class="involt-nav">

<a href="#page-1" class="involt-pagelink">First</a>

<a href="#page-2" class="involt-pagelink">Second</a>

<a href="#page-3" class="involt-pagelink">Third</a>

</nav>

</header>

<div page="page-1" class="involt-page home">First page</div>
<div page="page-2" class="involt-page">Second page</div>
<div page="page-3" class="involt-page">Third page</div>

Subpages and links

Second level of navigation can be made with involt-pagelink and involt-fullpage. They are opened on top of main navigation as full screen page. It's good to use them without main navigation when working on desktop app.

<a href="#page-4" class="involt-pagelink">Open new page</a>

The link will open the page:

<div page="page-4" class="involt-fullpage">Page content</div>

Fullpage can also have header:

<div page="page-4" class="involt-fullpage">

<div class="involt-appbar">

<div class="title-center">another page</div>

</div>


Page content

</div>

involt-fullback will hide the page

<div page="page-4" class="involt-fullpage">

<div class="involt-appbar">

<a href="#page-4" class="involt-fullback">Back</a>

<div class="title-center">Subpage</div>

</div>


Page content

</div>

Dialog

Dialog consists of button to open it, dialog box with its content and button to hide it. Dialog can be closed by pressing the background.
<a href="#dialog-1" class="involt-showdialog">Show dialog</a>
<div page="dialog-1" class="involt-dialog">

<a href="#dialog-1" class="involt-hidedialog">Hide dialog</a>

</div>

Side menu

involt-sidemenulink will open the side menu. If you wish you can make it a hamburger button with icon font.

<a href="" class="involt-sidemenulink">Menu</a>

The menu itself:

<div class="involt-sidemenu"></div>

Side menu can be closed by pressing the background.

Involt is ready to use Chrome packaged app template. You can use it to quickly create interactive object with software to hardware communication. The goal is to help designers to overcome the obstacles at prototyping stage.
