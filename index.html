An easy way to think outside the box.

Chrome based Arduino to HTML framework for designers.

Download

Involt is a bridge between easy to edit hardware interaction and HTML App interface.

No advanced coding

Create your app using only basic knowledge about HTML, CSS and Arduino. Instead of learning advanced coding you can focus on overall experience and quickly develop results.

Fast results with CSS

Using classes is really fast technique. Each UI element automatically communicates with Arduino via serial. Involt comes with editable UI kit to help you get started.

Easy development

Thanks to Chrome Apps you don't have to worry about building software from scratch. You don't need additional hardware to run Involt. It's only you and your project.


Three steps to create your project:

1.Install app
2.Edit Arduino sketch
3.Create the HTML layout
Read the getting started page

How does it work?

Involt translates CSS classes into functions. You only need to specify UI element, target pin and variables like desired values or their range.
<div class="ard button P5 255"> Click me </div>

For more advanced project you can use full features of JQuery and Chrome API. You can do the same thing with custom JQuery:

$(this).pinDefine("P5").sendValue(255);
